# terraform-multi-cloud-starter
Terraform Infrastructure as Code examples for AWS and Azure (learning-focused, production-inspired)
## Authentication

### AWS
Configure AWS credentials locally (for example):
- `aws configure`
or set environment variables (`AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`).

### Azure
Login with:
- `az login`

Terraform will use your Azure CLI session by default.
